# RealEstateAI — Launch Checklist

## Phase 1: Get the Site Live (Week 1)
- [ ] Review the site locally (`npm run dev`) and note any design/content fixes
- [ ] Swap placeholder tool data with real tools from `research/ai-tools-database.md` (50+ real AI tools)
- [ ] Add real affiliate links for tools with confirmed programs (Homesage.ai 40%, BirdDogBot 50%, Virtual Staging AI 30%, SofaBrain 20%, Lofty 10%)
- [ ] Sign up for affiliate programs (apply to each tool's partner/affiliate page)
- [ ] Buy a domain name (e.g., realestateaitools.com or similar)
- [ ] Deploy to Vercel (connect GitHub repo, deploy, add custom domain)
- [ ] Set up SSL (automatic with Vercel)

## Phase 2: SEO & Analytics (Week 1-2)
- [ ] Add Google Analytics 4 tracking
- [ ] Set up Google Search Console and submit sitemap
- [ ] Add proper meta tags, Open Graph images, and structured data (JSON-LD) for each page
- [ ] Generate and add a sitemap.xml
- [ ] Add robots.txt
- [ ] Set up canonical URLs on all pages
- [ ] Add alt text to all images/icons
- [ ] Verify site speed with Google PageSpeed Insights (target 90+)
